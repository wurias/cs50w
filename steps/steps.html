Variables (Variáveis)

Em Python, não é necessário explicitar o tipo na declaração da variável.

  • a = 30
  • b = 2.6
  • c = "Hello"
  • d = true

Formatting Strings

Para indicar que estamos usando strings formatadas, simplesmente adicionamos um 'f' antes das aspas.

print(f"Hello, {input("Name: ")}")
